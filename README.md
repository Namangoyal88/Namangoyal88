### 🧠 About Me

I'm an **AI/ML Engineer** who likes taking models past the notebook — from training to a deployable, monitored pipeline. My repos cover the full stack of a real ML workflow: deep learning architectures, classical ML from scratch, MLOps tooling (DVC, MLflow, Docker, GitHub Actions), and early experiments with LLM tooling like LangChain.

- 🔭 Currently exploring **end-to-end ML pipelines** with proper versioning, tracking, and CI/CD
- 🧩 Comfortable across the stack: **TensorFlow/Keras, scikit-learn, Flask, Docker, AWS EC2, GitHub Actions**
- 🌱 Currently digging into **LangChain / LCEL** and GenAI tooling
- ⚡ Fun fact: I've trained an LSTM on *Hamlet* — Shakespeare meets neural nets
